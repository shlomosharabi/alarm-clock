# WakeUp - שעון מעורר חכם 🔔

אפליקציית שעון מעורר עם משימות התעוררות, בנויה ב-React Native / Expo.

## הפעלה עם Expo Go (גרסה 54)

### דרישות מוקדמות
- Node.js 18+
- Expo Go 54.0.8 על האנדרואיד שלך
- חיבור לאותה רשת WiFi

### שלבי התקנה

```bash
# 1. התקן תלויות
npm install

# 2. הפעל את הפרויקט
npx expo start

# 3. סרוק את ה-QR עם Expo Go
```

### בעיות נפוצות

**שגיאת Reanimated?**
ודא ש-babel.config.js כולל את הפלאגין:
```js
plugins: ['react-native-reanimated/plugin']
```

**התראות לא עובדות ב-Expo Go?**
- הרשאות התראות חייבות להיות מאושרות ידנית
- ב-Expo Go חלק מתכונות ה-background מוגבלות
- לפונקציונליות מלאה יש לבנות APK עם `eas build`

## תכונות

### ⏰ שעון מעורר
- בחירת שעה ודקה עם גלגלת מגע
- שם מותאם אישית לכל שעון
- ימי חזרה (ראשון-שבת) או פעם אחת

### ⏱️ הגבלת משך צלצול
- הגדר כמה שניות/דקות השעון יצלצל לפני שיפסיק אוטומטית
- אפשרויות: 15 שניות / 30 שניות / 1 דקה / 2 דקות / 5 דקות

### 😴 נודניק (Snooze)
- הגדר זמן נודניק לכל שעון
- אפשרויות: כבוי / 5 / 10 / 15 / 20 דקות

### 🧠 משימות התעוררות
כדי לכבות את השעון, יש לפתור משימה:

| סוג | תיאור | דוגמה |
|-----|--------|-------|
| 🧮 מתמטיקה | חיבור, חיסור, כפל, חזקות | `7 × 8 = ?` |
| 📝 השלמת מילה | מילה עם אותיות חסרות | `מח_ב = ?` |
| 🔢 סדרות | מצא את המספר הבא | `2, 4, 8, ?` |

### רמות קושי
- 🟢 **קל** - פעולות בסיסיות, מילים קצרות
- 🟡 **בינוני** - כפל, מילים ארוכות, סדרות
- 🔴 **קשה** - חשבון מורכב, מילים מאתגרות, חזקות

## מבנה הקוד

```
alarm-app/
├── App.js                    # נקודת כניסה + navigation
├── index.js                  # registerRootComponent
├── app.json                  # הגדרות Expo
├── babel.config.js
├── package.json
├── assets/
│   ├── icon.png
│   ├── splash.png
│   └── adaptive-icon.png
└── src/
    ├── screens/
    │   ├── HomeScreen.js         # מסך ראשי - רשימת שעונים
    │   ├── EditAlarmScreen.js    # יצירה/עריכת שעון
    │   └── AlarmRingingScreen.js # מסך צלצול + משימה
    ├── store/
    │   └── alarmStore.js         # ניהול נתונים (AsyncStorage)
    └── utils/
        ├── taskGenerator.js      # יצירת משימות
        └── notifications.js      # ניהול התראות
```

## בניית APK לייצור

```bash
# התקן EAS CLI
npm install -g eas-cli

# הגדר את הפרויקט
eas build:configure

# בנה APK לאנדרואיד
eas build --platform android --profile preview
```
